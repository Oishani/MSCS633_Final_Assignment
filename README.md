# German Credit Bias Analysis

This repository contains a Jupyter Notebook that analyzes algorithmic bias in credit risk prediction using the German Credit dataset. The project evaluates how machine learning models can introduce or amplify disparities across protected attributes (e.g., gender, age) in financial decision-making.

Using IBM’s open-source AI Fairness 360 (AIF360) toolkit, a hands-on simulation of bias detection and mitigation was conducted on the German Credit Dataset - a benchmark dataset widely used in algorithmic fairness research. The simulation focuses on age bias in credit scoring decisions, applying the Reweighing pre-processing algorithm and measuring its effect across five fairness metrics.
